部署到 GitHub Pages

1. 创建 GitHub 仓库

  1. 登录 GitHub，点击右上角 + 号 → New repository
  2. 仓库名填写：你的用户名.github.io（例如：zhangsan.github.io）
  3. 选择 Public 公开仓库
  4. 点击 Create repository

2. 安装部署插件

3. 配置部署信息

确保 _config.yml 中的 deploy 部分已正确配置（见上文）。

4. 执行部署

首先检查ssh连接：

如果不正常，则需要先配置SSH连接：

步骤 1：生成 SSH 密钥（本地）

打开终端 / 命令行，执行以下命令（替换为你的 GitHub 邮箱）：

执行后，密钥会生成在默认路径：
Windows：C:\Users\你的用户名.ssh\id_ed25519.pub
Mac/Linux：~/.ssh/id_ed25519.pub

步骤 2：将公钥添加到 GitHub

1.复制公钥内容：

Windows：type C:\Users\你的用户名.ssh\id_ed25519.pub | clip
Mac/Linux：cat ~/.ssh/id_ed25519.pub | pbcopy

2.登录 GitHub → 点击右上角头像 → Settings → 左侧 SSH and GPG keys → New SSH key；
3.Title 填任意名称（如「Hexo 博客部署」），Key 粘贴复制的公钥内容 → 点击 Add SSH key

再次验证SSH连接：

SSH连接成功后再部署hexo

首次部署会要求输入 GitHub 账号和密码（或使用 Token）。

5. 访问你的博客

等待几分钟后，访问 https://你的用户名.github.io 即可看到你的博客！
